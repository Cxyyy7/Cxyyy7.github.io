在 JavaScript 里，对象如同神奇魔法屋，有 array、string、JSON、BOM 等重要成员。

  • array 像魔法口袋，能装各种类型数据，如 let myArray = [1, 'hello', {name: 'world'}];，通过索引获取元素，myArray[0] 可得 1。

  • string 似珠链，每个字符是珠子，如 let myString = 'JavaScript is amazing';，可获取长度 myString.length，也能提取部分，如 myString.substring(0, 4) 得 Java。

  • JSON 如严谨魔法书，用键值对存储数据，如 {"name": "Alice", "age": 25}，用于信息传递和解析。

  • BOM 像与浏览器相连的魔法城堡，可获取窗口信息，如 window.innerWidth，还能操作浏览器历史等。

  • 获取元素也关键，在 HTML 页面中，通过 document.getElementById 等方法，按元素 id 查找，如获取 <div id="myDiv"> 用 document.getElementById('myDiv')，之后可操作该元素。

  • 此外，JS - 对象还有其他特性和应用，共同构建丰富编程体验，在网页、应用开发等方面起关键作用。

    六、案例

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JS-对象-获取元素</title>
</head>
<body>
    <img id='h1' src="img/off.png" > <br><br>

    <div class="cls">江弘睿</div><br>
    <div class="cls">程序员</div><br>

    <input type="checkbox" name="hobby">电影
    <input type="checkbox" name="hobby">游戏
    <input type="checkbox" name="hobby">旅游
</body>

<script>
    //点亮灯泡 ： src 属性值
    var img =document.getElementById('h1');
    img.src='img/on.png';

    //将所有的 div 标签后加上：very good（红色字体）--<font color='red'><font>
    var divs=document.getElementsByTagName('div');
    for (let i = 0; i < divs.length; i++) {
        const div = divs[i];
        div.innerHTML+="<font color='red'> very good<font>";
    }

    //使复选框呈现被选中状态
    var ins=document.getElementsByName('hobby');
    for (let i = 0; i < ins.length; i++) {
        const check = ins[i];
        check.checked=true;
    }
</script>
</html>
